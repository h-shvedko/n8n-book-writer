## Kapitel 1: HTML & Semantic Markup

**Frage 1:** What is the purpose of the DOCTYPE declaration in an HTML5 document?

A) To specify the character encoding
B) To define the document's language
C) To inform the browser about the HTML version
D) To link external stylesheets

> **Antwort:** C
> *The DOCTYPE declaration informs the browser about the HTML version being used, ensuring correct rendering.*

---

**Frage 2:** Why are semantic HTML elements important?

A) They improve page load speed
B) They enhance accessibility and SEO
C) They reduce server load
D) They increase browser compatibility

> **Antwort:** B
> *Semantic elements provide clear meaning to the content, aiding accessibility tools and search engines.*

---

**Frage 3:** How do you correctly associate a label with an input field?

A) By placing the label inside the input tag
B) By using the 'for' attribute in the label
C) By using the 'name' attribute in the input
D) By using the 'class' attribute in the label

> **Antwort:** B
> *The 'for' attribute in a label should match the 'id' of the input field to associate them correctly.*

---

**Frage 4:** Which HTML5 input type would you use for a date selection?

A) text
B) email
C) date
D) number

> **Antwort:** C
> *The 'date' input type provides a date picker interface for selecting dates.*

---

**Frage 5:** What is a potential pitfall of relying solely on HTML5 validation?

A) It may not work on all browsers
B) It increases server load
C) It requires additional CSS
D) It slows down page rendering

> **Antwort:** A
> *HTML5 validation is not consistently supported across all browsers, so server-side validation is also necessary.*

## Kapitel 2: CSS & Responsive Design

**Frage 1:** What is the primary purpose of using CSS selectors?

A) To define the layout of a webpage
B) To identify HTML elements for styling
C) To create animations
D) To manage server-side scripts

> **Antwort:** B
> *CSS selectors are used to identify HTML elements to which styles will be applied.*

---

**Frage 2:** Which CSS layout technique is best suited for creating a two-dimensional grid?

A) Flexbox
B) Grid
C) Float
D) Inline-block

> **Antwort:** B
> *CSS Grid is specifically designed for creating two-dimensional layouts, allowing for both rows and columns.*

---

**Frage 3:** How do media queries contribute to responsive design?

A) By changing the server configuration
B) By adjusting styles based on device properties
C) By optimizing images
D) By managing database connections

> **Antwort:** B
> *Media queries allow styles to be applied conditionally based on device characteristics like screen size.*

---

**Frage 4:** Why is a mobile-first design approach beneficial?

A) It increases server load
B) It improves performance on mobile devices
C) It reduces code complexity
D) It enhances desktop experience

> **Antwort:** B
> *A mobile-first approach ensures that the design is optimized for mobile devices, which are commonly used.*

---

**Frage 5:** What is the specificity hierarchy in CSS?

A) Inline styles > ID selectors > Class selectors > Element selectors
B) Element selectors > Class selectors > ID selectors > Inline styles
C) Class selectors > Element selectors > ID selectors > Inline styles
D) ID selectors > Inline styles > Class selectors > Element selectors

> **Antwort:** A
> *The specificity hierarchy determines which CSS rule is applied when multiple rules target the same element.*
